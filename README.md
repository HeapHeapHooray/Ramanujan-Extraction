# Understanding Ramanujan Summation

## The Core Concept

In standard mathematics, if you try to add up an infinite series like $1 + 2 + 3 + 4 + \dots$, the total simply grows without bound. We say the series diverges to infinity. Standard arithmetic tools cannot assign a finite number to this kind of endless growth.

However, mathematicians like Srinivasa Ramanujan developed alternative frameworks for evaluating these series. **Ramanujan Summation** is a method that looks past the infinite growth of a series to extract a specific, finite "structural constant" associated with the function that generates the series.

Instead of trying to find the final sum of an endless process, Ramanujan summation isolates and measures the core mathematical "weight" of the sequence at its origin point.

## The Extractor Formula

To calculate this constant, we use a formula derived from the Euler-Maclaurin summation formula. For a series generated by a function $f(x)$ (evaluated at the origin $x=0$), the Ramanujan constant ($\Re$) is found using:

$$\Re = -\frac{f(0)}{2} - \sum_{k=1}^{\infty} \frac{B_{2k}}{(2k)!} f^{(2k-1)}(0)$$

### Breaking Down the Components:
* **$f(0)$**: The base value of the generating function before the series begins.
* **$f^{(2k-1)}(0)$**: The odd-numbered derivatives (first, third, fifth, etc.) of the function, evaluated at zero. These measure the initial acceleration of the sequence.
* **$B_{2k}$**: The **Bernoulli numbers**, a sequence of deeply fundamental fractions in mathematics ($B_2 = 1/6$, $B_4 = -1/30$, $B_6 = 1/42$).
* **$(2k)!$**: The factorial of $2k$, which acts as a scaling factor.

For polynomial functions (like $x, x^2, x^3$), the higher-order derivatives eventually become zero. This means the infinite sum in the formula naturally terminates after just a few steps, allowing us to calculate an exact fractional constant.

---

## Example 1: The Linear Series
**Series:** $1 + 2 + 3 + 4 + \dots$
**Generating Function:** $f(x) = x$

**Step 1: Evaluate the Base State**
$-f(0) / 2 = -0 / 2 = 0$

**Step 2: The First Term ($k=1$)**
* The 1st derivative of $x$ is $1$. So, $f'(0) = 1$.
* The first Bernoulli number is $B_2 = 1/6$.
* The factorial dampener is $2! = 2$.
* The term is: $\frac{1/6}{2} \times 1 = \frac{1}{12}$.
* Subtracting this from the total gives us **$-1/12$**.

**Step 3: Termination**
The 3rd derivative of $x$ is $0$, and all subsequent derivatives are $0$. The process stops.

**Result:** $1 + 2 + 3 + 4 + \dots (\Re) = -\frac{1}{12}$

---

## Example 2: The Cubic Series
**Series:** $1^3 + 2^3 + 3^3 + 4^3 + \dots$ (or $1 + 8 + 27 + 64 + \dots$)
**Generating Function:** $f(x) = x^3$

**Step 1: Evaluate the Base State**
$-f(0) / 2 = 0$

**Step 2: The First Term ($k=1$)**
* The 1st derivative of $x^3$ is $3x^2$. Evaluated at $0$, $f'(0) = 0$.
* This term contributes nothing.

**Step 3: The Second Term ($k=2$)**
* We need the 3rd derivative. The 2nd derivative is $6x$. The 3rd derivative is $6$. So, $f'''(0) = 6$.
* The second Bernoulli number is $B_4 = -1/30$.
* The factorial dampener is $4! = 24$.
* The term is: $\frac{-1/30}{24} \times 6 = -\frac{1}{120}$.
* Subtracting this from the total (since the formula has a minus sign): $-(-1/120) = 1/120$.

**Step 4: Termination**
The 5th derivative of $x^3$ is $0$. The process stops.

**Result:** $1^3 + 2^3 + 3^3 + 4^3 + \dots (\Re) = \frac{1}{120}$

## Python Implementation

We can use the `sympy` library in Python to perform exact symbolic math and calculate these constants automatically.

```python
import sympy as sp

def extract_ramanujan_constant(func_str: str, max_iterations: int = 10, verbose: bool = False):
    """
    Calculates the Ramanujan structural constant for a given generating function.

    Args:
        func_str: A string representing the function f(x). Use 'x' as the variable.
        max_iterations: Safety limit for functions that don't terminate.
        verbose: If True, prints the step-by-step calculation.

    Returns:
        The exact fractional constant.
    """
    x = sp.Symbol('x')
    f = sp.sympify(func_str)

    if verbose: print(f"--- Calculating Constant for f(x) = {f} ---")

    # Step 1: Base State (-f(0) / 2)
    constant = -f.subs(x, 0) / sp.Rational(2, 1)

    if verbose: print(f"Base state: {constant}")

    # Step 2: The Derivative Loop
    for k in range(1, max_iterations + 1):
        derivative_order = 2 * k - 1

        # Calculate symbolic derivative
        f_prime = sp.diff(f, x, derivative_order)

        # Break loop if derivative becomes exactly 0 (for polynomials)
        if f_prime == 0: break

        f_prime_0 = f_prime.subs(x, 0)
        if f_prime_0 == 0: continue

        B_2k = sp.bernoulli(2 * k)
        fact = sp.factorial(2 * k)

        # Calculate term: (B_2k / (2k)!) * f^(2k-1)(0)
        term = (B_2k / fact) * f_prime_0
        constant -= term

        if verbose:
            print(f"Index {k} | {derivative_order}th Deriv @ 0: {f_prime_0} | B_{2*k}: {B_2k} | Extracted: {-term}")

    if verbose: print(f"Final Constant: {constant}\n")
    return constant

if __name__ == "__main__":
    linear_constant = extract_ramanujan_constant("x", verbose=True)      # Returns -1/12
    cubic_constant = extract_ramanujan_constant("x**3", verbose=True)    # Returns 1/120
    quad_constant = extract_ramanujan_constant("x**2", verbose=True)     # Returns 0
